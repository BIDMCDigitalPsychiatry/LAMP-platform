Customize Emotions on a DBT Diary Card

  1. Log in to the dashboard and navigate to the Activities tab.
  2. Click the [+ Add] button at the top right of the list and select DBT Diary Card.
  3. Create a title for the client's DBT Diary Card.
  4. Add the life worth living goal.
  5. Add target behaviors and target ineffective behavoirs.
  6. Add an emotion by selecting the red [+ ADD ITEM] text.
  7. Type the emotion in the text box.

  1. Save the customized diary card by clicking the blue Save button in the bottom right-hand corner.
