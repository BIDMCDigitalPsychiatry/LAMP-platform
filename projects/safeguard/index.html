Project Showcase

Safeguard

Safeguard is a life skills training program for U.S. Army soldiers, delivered through a customized mindLAMP experience built to support group-based program delivery. The features below help administrators organize app content, personalize each cohort’s experience, and guide participants to the activities, resources, and communication channels most relevant to their group.

Home Page

The Home Page feature gives mindLAMP users a central landing page that can highlight the most important content for a specific app account or group. Administrators can customize the page with a banner, image, linked activity, quick-action button, favorites or selected activities, calendar view, and progress/streak information. This helps users quickly find key activities while giving each mindLAMP implementation a more organized and tailored entry point.

Use Cases

The Home Page can be used in mindLAMP to orient participants toward the most important activities, such as a daily check-in, study task, or core intervention module. It can also bring together frequently used content, favorited activities, calendar-based tasks, and progress feedback in one place. For multi-group deployments, administrators can create different home page experiences for different cohorts while keeping the app structure consistent.

Configuration Steps

Access Home Settings

  1. From the administrator dashboard, go to Users and select the relevant app account or group.
  2. Click the Home Settings gear icon for the group whose home page should be customized.

Banner Settings

  1. Add the Banner Title and Banner Subheading that should appear at the top of the home page.
  2. Select the activity that should open when users tap the banner.
  3. Upload or select a Banner Image to display in the banner area.

Small Action Button Settings

  1. Enter the text that should appear on the small action button.
  2. Select the activity that should open when users tap the button.

Home Page Activities

  1. Choose whether the home page should show Favorites, a Custom set of activities, or None.
  2. If using Custom, click Add Activity and select the activities that should appear on the home page.

Featured Activity Settings

  1. Choose whether a specific activity should be highlighted more prominently on the home page.
  2. Select the activity that should appear as the featured activity.

Home Page Streak Settings

  1. Choose the streak type that should be displayed on the home page, such as daily, weekly, monthly, or none.
  2. Add the streak title and description that users should see.

Survey Group Settings

  1. Use Survey Group Settings to group related surveys together for home page tracking.
  2. Use Ungrouped Surveys to identify surveys that should remain outside grouped tracking.
